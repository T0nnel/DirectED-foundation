# DirectEd Development Foundation

A modern web application for the DirectEd Development Foundation, empowering Africa's next generation of tech leaders through world-class training and remote paid internships.

## Project Overview

This website showcases the DirectEd Development Foundation's mission, programs, success stories, and opportunities for students across Africa.

## Technologies Used

This project is built with:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Beautiful component library
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Smooth animations
- **i18next** - Multi-language support (English, French, Spanish, Swahili)
- **React Router** - Client-side routing
- **Supabase** - Backend and authentication

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd DirectED-foundation

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build in development mode
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

Build the project for production:

The built files will be in the `dist` directory, ready for deployment to any static hosting service.
